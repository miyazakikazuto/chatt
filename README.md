# Chatt

Aplikasi chat sederhana berbasis web.

## Fitur

- Antarmuka chat yang simpel dan responsif
- Dua versi halaman (\`index.html\` dan \`index2.html\`)
- Siap di-deploy ke GitHub Pages

## Teknologi

- HTML + CSS + JavaScript
- GitHub Pages

## Live Demo

[https://miyazakikazuto.github.io/chatt/](https://miyazakikazuto.github.io/chatt/)

## Cara Pakai

1. Clone repositori
2. Buka \`index.html\` di browser
3. Atau deploy ke GitHub Pages untuk akses online
